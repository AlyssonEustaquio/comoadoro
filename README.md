# 🍳 Controle de Receitas e Custos

Um sistema web simples e eficiente para calcular o custo de receitas e ingredientes.

## ✨ Funcionalidades

- 📦 **Cadastro de Produtos**: Adicione ingredientes com preço e quantidade
- 📝 **Criação de Receitas**: Crie suas receitas e veja o custo total automaticamente
- ⚡ **Recálculo Automático**: Quando você muda o preço de um produto, todas as receitas são recalculadas instantaneamente
- 💾 **Armazenamento Local**: Todos os dados são salvos no seu navegador (100% offline)
- 📊 **Cálculo de Custo Unitário**: Veja o custo por unidade de rendimento

## 🚀 Como Usar

1. Acesse: https://alyssoneustaquio.github.io/comoadoro/

2. **Aba Produtos**: Cadastre seus ingredientes
   - Nome do produto
   - Preço total
   - Quantidade
   - Unidade de medida (kg, g, l, ml, unidade, etc)

3. **Aba Receitas**: Crie suas receitas
   - Nome da receita
   - Rendimento (quantas unidades produz)
   - Selecione ingredientes e quantidades
   - O custo total é calculado automaticamente!

## 💡 Exemplo

Se você cadastrar:
- Farinha de trigo: R$ 5,00 / 1kg
- Ovos: R$ 8,00 / 12 unidades

E criar uma receita "Bolo" com:
- 500g de farinha
- 3 ovos
- Rendimento: 8 fatias

O sistema calcula:
- Custo total: R$ ~4,50
- Custo por fatia: R$ ~0,56

## 🔒 Privacidade

Todos os dados são armazenados **apenas no seu navegador** usando localStorage. Nenhuma informação é enviada para servidores.

## 📱 Compatibilidade

- ✅ Desktop (Chrome, Firefox, Safari, Edge)
- ✅ Mobile (iOS Safari, Chrome Mobile)
- ✅ Funciona completamente offline
